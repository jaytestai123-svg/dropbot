Commands

Simple. Powerful.

Everything you need, nothing you don't.

/gstart
Start a new giveaway. Set prize, duration, winners, requirements, and bonus roles.
/gend
End a giveaway early and trigger the slot machine reveal immediately.
/greroll
Re-spin the slot for a new winner. Auto-selects the last giveaway — no ID needed.
/glist
Show all active giveaways in your server with entry counts.
/gdelete
Cancel a giveaway without picking winners.
/gsettings
Configure DM winners, default channel, and server preferences.
